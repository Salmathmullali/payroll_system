 <h1>Enterprise Payroll ERP System</h1><br><br>A robust, logic-driven Payroll Management System built to automate complex financial calculations for organizations. <br>This system replaces manual spreadsheets with a secure, scalable backend architecture. <br><br><h3>Core Features</h3><br>Automated Salary Processing: <br>Real-time calculation of Gross and Net salaries.<br><br>Indian Statutory Compliance: <br>Built-in logic for PF (Provident Fund), ESI, and Professional Tax deductions.<br><br>Dynamic Allowances: <br>Automated handling of DA (Dearness Allowance) and HRA (House Rent Allowance) based on employee grade.<br><br>Payroll Dashboard: <br>A clean UI to track monthly disbursements and employee records.<br><br>Reporting: <br>Generates detailed salary slips and financial summaries.<br><br><h3>🛠️ Tech StackBackend:</h3><br> Python / DjangoDatabase: PostgreSQL / SQLite (for development)<br>Frontend: HTML5, CSS3, JavaScript (Django Templates)<br>APIs: RESTful architecture for data consistency.<br><br><h2>Business Logic Implementation</h2><br>One of the key challenges solved in this project was the precision of financial arithmetic. <br><br>The system handles:<br>Deductions: <br>Logic-based calculation for $PF = 12\%$ of Basic Pay.<br>Taxation: <br>Automated slabs for Professional Tax based on regional regulations.<br>Security: <br>Role-based access control (RBAC) ensuring only HR/Admins can view sensitive financial data.
